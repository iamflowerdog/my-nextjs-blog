# 说明

这个网站构建于 [Tailwind CSS](https://tailwindcss.com) 和 [Next.js](https://nextjs.org).

## 如何运行在你的机器

先安装依赖

```bash
npm install
```
然后cd到你的目录下

```bash
npm run dev
```

最后, 打开 [http://localhost:3000](http://localhost:3000)

改动代码，自动重新部署。
