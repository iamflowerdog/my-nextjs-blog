## **先在Mac上安装Ruby环境**

首先，打开终端。

然后输入以下命令来检查当前系统上是否已经安装了Ruby：

```bash
ruby -v
```

如果已经安装了，会显示当前安装的版本号。

如果没有安装Ruby，可以使用Homebrew进行安装。

使用Homebrew安装Ruby，输入命令：

```bash
brew install ruby
```

安装完成后，再次输入 

```bash
ruby -v 
```

会显示当前安装的版本号。

如果要升级Ruby版本，可以使用

```bash
brew upgrade ruby
```

来完成。



## **安装Cocoapods**

```bash
sudo gem install cocoapods
```

这将在您的计算机上安装CocoaPods。


## **使用CocoaPods向Xcode项目安装软件包**

### **推荐软件包：SwifterSwift**

<Image src={swifterswift} alt="" />

https://github.com/SwifterSwift/SwifterSwift

SwifterSwift 是500 多个本机 Swift 扩展的集合，具有方便的方法、语法糖和针对各种原始数据类型、UIKit 和 Cocoa 类的性能改进——超过 500 个合 1——适用于 iOS、macOS、tvOS、watchOS 和 Linux。

创建或打开一个Xcode项目。

在终端中使用cd + 拖拽文件夹的方式，进入到项目目录下，输入：

```bash
pod init 
```

这将在您的项目目录中创建一个名为“Podfile”的文件。
使用编辑器打开Podfile，在其中添加您需要使用的第三方库和框架。例如：

```
pod 'SwifterSwift'
```

这表示安装'SwifterSwift'库

保存Podfile。

在终端中输入：

```bash
pod install 
```

这将安装您在Podfile中指定的所有依赖项。

打开您的项目，从现在开始，您应该使用<b>.xcworkspaced</b>文件而不是.xcodeproj文件进行开发。
